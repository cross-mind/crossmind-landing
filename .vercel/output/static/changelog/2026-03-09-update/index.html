Multi-Session Copilot & Workspace Improvements

Run multiple copilot sessions simultaneously, get notified when they complete, and see file changes more clearly

✨ Features 🔧 Improvements

We’ve shipped major improvements to how you work with multiple sessions and track your agent’s file changes.

Run Multiple Copilot Sessions at Once

You can now have multiple copilot sessions running simultaneously. Previously, starting a new copilot session would pause or archive the previous one. Now they all run concurrently, and you can easily switch between them.

Why it matters: You can now work on multiple conversations or problems in parallel without losing context. Perfect for multitasking - kick off a research task in one session while continuing your main conversation in another.

Instant Notifications When Sessions Complete

Added smart notification indicators for running and completed sessions. You’ll see:

  • Pulsing blue dots on sessions that are currently running
  • Solid green dots on completed sessions you haven’t viewed yet
  • The dots disappear once you check the completed session

Why it matters: No more constantly checking back to see if your autonomous task finished. The visual indicators let you focus on other work and return exactly when something’s ready for your review.

See Exactly What Your Agent Changed

File changes are now automatically captured at the end of every session, even if the agent didn’t explicitly commit them. The Changes tab now shows a complete diff of everything modified during the session.

Why it matters: Better visibility into what your agent actually did. You can review every file modification with proper before/after diffs, making it easier to understand changes and catch issues early.

Cleaner Workspace Navigation

Refined the layout and detail views across the workspace UI. Session switching, artifact browsing, and task navigation all feel more consistent and polished.

Why it matters: Small refinements add up. The interface is more predictable, reducing friction when you’re jumping between different parts of your workspace.
